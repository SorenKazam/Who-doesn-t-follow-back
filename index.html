Instructions how to use!

How to use this website!

  1. Your Instagram needs to be in English
  2. This Website is made to work only in Desktop! Accessing it on mobile may result in some bugs
  3. Open your Instagram profile
  4. Click on "Followers" to display your followers list
  5. Copy from top to bottom all the people you have inside your "Followers" list!
  6. Paste everything inside the first list
  7. Do the same thing with the "Following" list from your profile
  8. Click on "Find out!" button
  9. If you followed the Instructions step-by-step, you should now see the "Results" table bellow!

Some troobleshotting

  • If you are reciving "List X is empty" is because you didnt paste the list inside the textarea
  • If you are reciving "No followers found" message, is because the list you paste isnt in the right format!:
    • Make sure when you paste your followers or following list into the textarea that you have every person in the list and make sure you have "personID's profile picture" like this in the list! make sure that the "..'s profile picture" is in english! if it is not in english the code will not work! Make sure your Instagram is set to english!
